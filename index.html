Трансформация токов несимметричных КЗ через силовые трансформаторы

Аналитический вывод: Двухфазное КЗ (B-C) за трансформатором Δ/Y-0

1. Исходные уравнения несимметричного режима
При возникновении двухфазного замыкания между фазами $B$ и $C$ на выводах вторичной обмотки (НН), согласно первому и второму законам Кирхгофа, фиксируются следующие граничные условия для токов: $$\dot{I}_{A2} = 0; \quad \dot{I}_{B2} = -\dot{I}_{C2}$$ При этом модуль тока КЗ выражается через ток базисного трехфазного повреждения как: $$I_{КЗ2}^{(2)} = \frac{\sqrt{3}}{2} I_{КЗ2}^{(3)}$$
2. Закон трансформации токов в фазные обмотки ВН
Поскольку первичная обмотка соединена в треугольник, фазные токи внутри него ($\dot{I}_{AB1}, \dot{I}_{BC1}, \dot{I}_{CA1}$) пропорциональны токам в соответствующих фазах вторичной звезды с учетом коэффициента трансформации $k_{тт}$: $$\dot{I}_{AB1} = \frac{\dot{I}_{A2}}{\sqrt{3} \cdot k_{тт}} = \frac{I_{КЗ2}^{(3)}}{2 \cdot k_{тт}}$$ $$\dot{I}_{BC1} = 0; \quad \dot{I}_{CA1} = -\frac{I_{КЗ2}^{(3)}}{2 \cdot k_{тт}}$$
3. Расчет линейных токов первичной сети
Линейные токи, поступающие из сети высокого напряжения в трансформатор, определяются разностью соответствующих фазных токов в вершинах треугольника: $$ \begin{cases} \dot{I}_{A1} = \dot{I}_{AB1} - \dot{I}_{CA1} = \frac{I_{КЗ2}^{(3)}}{2 k_{тт}} - \left(-\frac{I_{КЗ2}^{(3)}}{2 k_{тт}}\right) = \frac{I_{КЗ2}^{(3)}}{k_{тт}} \\ \dot{I}_{B1} = \dot{I}_{BC1} - \dot{I}_{AB1} = 0 - \frac{I_{КЗ2}^{(3)}}{2 k_{тт}} = -\frac{I_{КЗ2}^{(3)}}{2 k_{тт}} \\ \dot{I}_{C1} = \dot{I}_{CA1} - \dot{I}_{BC1} = -\frac{I_{КЗ2}^{(3)}}{2 k_{тт}} - 0 = -\frac{I_{КЗ2}^{(3)}}{2 k_{тт}} \end{cases} $$
4. Результирующие коэффициенты формы
Связывая токи первичной стороны с фактическим током двухфазного КЗ на вторичной стороне ($I_{КЗ2}^{(2)}$), получаем окончательные модульные зависимости: $$I_{A1} = \frac{2}{\sqrt{3}} \cdot \frac{I_{КЗ2}^{(2)}}{k_{тт}} \approx 1.155 \cdot \frac{I_{КЗ2}^{(2)}}{k_{тт}}$$ $$I_{B1} = I_{C1} = \frac{1}{\sqrt{3}} \cdot \frac{I_{КЗ2}^{(2)}}{k_{тт}} \approx 0.577 \cdot \frac{I_{КЗ2}^{(2)}}{k_{тт}}$$

1. Двухфазное КЗ за трансформатором Y/Y-0

Модуль тока двухфазного КЗ в общем виде определяется выражением:

\[I_{КЗ}^{(2)} = |\dot{I}_A| = |\dot{I}_C| = \frac{\sqrt{3} U_ф}{\sqrt{(r_1+r_2)^2 + (x_1+x_2)^2}} = \frac{U_л}{\sqrt{(r_1+r_2)^2 + (x_1+x_2)^2}}\]

При условии равенства полных сопротивлений прямой и обратной последовательностей (\(Z_1 = Z_2\)):

\[I_{КЗ}^{(2)} = \frac{U_л}{2\sqrt{r_1^2 + x_1^2}} = \frac{\sqrt{3}}{2} I_{КЗ}^{(3)}\]

Токи на стороне ВН при двухфазном КЗ за трансформатором со схемой соединения Y/Y-0 распределяются следующим образом:

\[\begin{cases} \dot{I}_{A1} = \frac{\sqrt{3}}{2} \cdot \frac{I_{КЗ2}^{(3)}}{k_{тт}} \\ \dot{I}_{B1} = 0 \\ \dot{I}_{C1} = -\dot{I}_{A1} \end{cases}\]

2. Однофазное КЗ за трансформатором Y/Y-0

Модуль тока однофазного КЗ определяется через симметричные составляющие:

\[I_{КЗ}^{(1)} = |\dot{I}_A| = \frac{3 U_{ф2}}{\sqrt{(r_1+r_2+r_0)^2 + (x_1+x_2+x_0)^2}} = \frac{\sqrt{3} U_{л.2}}{\sqrt{(r_1+r_2+r_0)^2 + (x_1+x_2+x_0)^2}}\]

При условии, что \(Z_1 = Z_2\), ток КЗ на вторичной стороне равен:

\[I_{КЗ2}^{(1)} = \frac{\sqrt{3} U_{л2}}{\sqrt{(2r_1+r_0)^2 + (2x_1+x_0)^2}}\]

Непосредственно для КЗ за трансформатором Y/Y-0 с учетом его параметров:

\[I_{КЗ2.Y/Y-0}^{(1)} = \frac{\sqrt{3} U_{л2}}{\sqrt{(2r_{1.т}+r_{0.т})^2 + (2x_{1.т}+x_{0.т})^2}} = \frac{\sqrt{3} U_{л2}}{Z_{т.Y/Y-0(1)}}\]

Трансформация токов на сторону высшего напряжения (ВН) с учетом коэффициента трансформации \(k_{тт}\):

  • Ток в фазе «А» на стороне ВН: \[\dot{I}_{A.1} = \frac{\dot{I}_1 + \dot{I}_2}{k_{тт}} = \frac{2\dot{E}_{ф.2}}{k_{тт} (Z_1 + Z_2 + Z_0)} = \frac{2}{3} \cdot \frac{I_{КЗ2.Y/Y-0}^{(1)}}{k_{тт}}\]
  • Токи в фазах «B» и «С» на стороне ВН (где \(a = e^{j120^\circ}\) — оператор фазы): \[\dot{I}_{B.1} = \frac{a^2 \dot{I}_1 + a \dot{I}_2}{k_{тт}} = \frac{-\dot{E}_{ф.2}}{k_{тт} (Z_1 + Z_2 + Z_0)} = -\frac{1}{3} \cdot \frac{I_{КЗ2.Y/Y-0}^{(1)}}{k_{тт}}\] \[\dot{I}_{C.1} = \frac{a \dot{I}_1 + a^2 \dot{I}_2}{k_{тт}} = \frac{-\dot{E}_{ф.2}}{k_{тт} (Z_1 + Z_2 + Z_0)} = -\frac{1}{3} \cdot \frac{I_{КЗ2.Y/Y-0}^{(1)}}{k_{тт}}\]

Схема соединения обмоток Y/Z-11 (Звезда-Зигзаг с заземленной нейтралью)

Каждая полуобмотка низшего напряжения (НН) индуктивно связана с обмоткой высшего напряжения (ВН) другой фазы. Из-за встречно-последовательного включения полуобмоток коэффициент трансформации по отношению к одной секции зигзага составляет \(k_{т.об.Y/Z} = \frac{U_ф}{U_ф / \sqrt{3}} = \sqrt{3}\).
